# DecodeLabs Internship - Backend API

## Project Overview
This project is a simple REST API built using Node.js and Express as part of the DecodeLabs Full Stack Development Internship.

## Features
- GET /users endpoint
- POST /users endpoint
- JSON response handling
- Express.js backend server

## Technologies Used
- Node.js
- Express.js
- JavaScript
